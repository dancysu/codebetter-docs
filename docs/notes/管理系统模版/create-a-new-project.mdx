# 创建新项目

-----

**步骤 1：创建新项目**

首先,我们将使用 [Vite](https://cn.vitejs.dev/guide/#scaffolding-your-first-vite-project) 来创建一个新的项目。请打开您的终端,并运行以下命令：

```bash
pnpm create vite
```

-----

**步骤 2：配置项目**

在创建过程中,您将被提示输入项目信息。请按照以下建议进行选择：

```
✔ Project name: … vue3-admin-template
✔ Select a framework: › Vue
✔ Select a variant: › TypeScript
```
----

**步骤3：进入项目并启动开发服务器**

项目配置完成后,进入您的新项目目录,并执行以下命令来安装依赖项并启动开发服务器：

```bash
cd vue3-admin-template
pnpm install
pnpm run dev
```

现在,您的项目环境已经搭建完成,您可以开始开发了。

---

**步骤 4：初始化项目结构**

为了确保项目的高效和有序开发,以下是推荐的项目结构：

```bash
.
├─ public              # 静态资源目录,直接复制到构建输出目录
├─ mock                # 存放模拟数据和接口的目录
├─ src
│  ├─ common           # 公共代码和资源目录
│  │  ├─ apis          # API接口目录
│  │  ├─ assets        # 资源目录,包含图标、图片和样式文件等
│  │  │  ├─ icons           
│  │  │  ├─ images          
│  │  │  └─ styles          
│  │  ├─ components    # 可复用组件目录       
│  │  ├─ constants     # 常量目录         
│  │  ├─ hooks         # 自定义钩子函数目录             
│  │  ├─ plugins       # 插件目录           
│  │  ├─ types         # TS类型定义文件目录             
│  │  └─ utils         # 工具函数目录             
│  ├─ http             # HTTP目录                       
│  ├─ layouts          # 页面布局组件目录              
│  ├─ pages            # 应用页面目录                 
│  ├─ pinia            # Pinia状态管理目录                
│  ├─ router           # 路由配置目录               
│  ├─ App.vue          # 应用根组件              
│  └─ main.ts          # JS入口文件,初始化Vue实例及挂载DOM            
├─ index.html          # 应用入口文件
├─ package.json        # 项目依赖管理和脚本配置文件
├─ README.md           # 项目说明文档
├─ tsconfig.app.json   # 应用TS配置文件
├─ tsconfig.json       # 项目TS配置文件
├─ tsconfig.node.json  # Node.js环境下的TS配置文件
└─ vite.config.ts      # Vite配置文件
```
