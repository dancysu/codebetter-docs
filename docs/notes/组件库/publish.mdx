# 发布组件库

----

**步骤 1：创建 npm 账号**

首先,访问 [npm官方网站](https://www.npmjs.com/) 并注册一个账号。注册完成后,在 npm 平台中确认是否存在和 `package.json` 文件的 `name` 属性相同名称的包名。如果存在,则需要修改 `name` 属性的值,确保其是唯一的。

-----

**步骤 2：设置发布源**

在项目开发过程中,你可能使用了如淘宝 npm 镜像等非官方镜像源。为了确保能够将包发布到 npm 官方平台,你需要将 npm 的发布源切换回官方源：

```bash
npm config set registry https://registry.npmjs.org/
```

----

**步骤 3：登录 npm**

在终端执行以下命令时,会提示你按下 `enter` 键,使用浏览器登录：

```bash
npm login
```

----

**步骤 4：发布包**

```bash
npm publish
```

----

**步骤 5：验证发布结果**

发布成功后,访问 [npm官方网站](https://www.npmjs.com/) 并使用你的账号登录,检查你的包是否已经成功发布。
