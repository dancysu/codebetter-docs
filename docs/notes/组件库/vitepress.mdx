# Vitepress

## 初始化文档

**步骤 1：进入文档目录**

```bash
cd docs
```

**步骤 2：初始化项目**

这条命令初始化一个新的 npm 项目,并创建一个 package.json 文件。

```bash
pnpm init
```

**步骤 3：安装 vitepress**

这条命令将 [vitepress](https://vitepress.dev/zh/) 作为开发依赖安装到您的项目中。

```bash
pnpm add -D vitepress
```

**步骤 4：启用设置向导**

这条命令启动 VitePress 的设置向导,帮助您配置 VitePress。

```bash
pnpm vitepress init
```

## 配置导航栏

```js
import { defineConfig } from "vitepress";

export default defineConfig({
  title: "Suit组件库",
  description: "A Vue3 UI library",
  themeConfig: {
    nav: [
      { text: "指南", link: "/guide/" }, // [!code ++]
      { text: "组件", link: "/components/" }, // [!code ++]
    ],

    sidebar: {},

    socialLinks: [
      { icon: "github", link: "https://github.com/vuejs/vitepress" },
    ],
  },
});
```
