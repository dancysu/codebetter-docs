# BEM命名规范

BEM（Block Element Modifier）命名规则是一种CSS命名的方法论。



命名格式示例

```
suit-tabs__item--size_mini
```

:::details 命名规则

`<block_name>__<element_name>--<modifier-name>_<modifier_value>`

- 所有组件的命名均使用小写字母,复合词使用短横线 `-` 连接

- Block 和 Element 之间使用双下划线 `__` 连接

- Modifier 与 Block/Element 使用双短横线 `--` 连接

- `modifier-name` 和 `modifier_value` 之间使用下划线 `_` 连接

:::
