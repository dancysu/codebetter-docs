# 安装
**安装Rustup**
安装 Rust 的主要方式是通过 Rustup 这一工具,它既是一个 Rust 安装器又是一个版本管理工具。
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs  | sh
```
**配置环境变量**
```bash
# 打开配置文件
vim ~/.zshrc
# 添加以下内容
export PATH="$HOME/.cargo/bin:$PATH"
# 保存文件并运行以下命令以使更改生效：
source ~/.zshrc
```
**验证安装**
```bash
rustc --version
# 结果显示：rustc x.y.z (commit-hash yyyy-mm-dd)
```
**更新Rust**
```bash
rustup update
```
**卸载Rust**
```bash
rustup self uninstall
```
**打开本地文档**
安装 rust 的时候,还会在本地安装文档,可离线浏览。运行以下命令可在浏览器打开本地文档
```bash
rustup doc
```
