# 猜数游戏
目标：生成一个1到100之间的随机数,提示玩家输入一个猜测,猜完之后,程序会提示猜测是太小了还是太大了,如果猜测正确,那么打印出一个庆祝信息,程序退出。
创建项目
```bash
cargo new guessing_game
```
运行项目
运行一下项目,看看是否有问题
```bash
cargo run 
```
接下来就可开始写代码了
```rust
// 引入标准输入输出库
use std::io; 
// 主函数
fn main() {
    // 打印游戏开始提示
    println!("猜数游戏!");
    println!("猜测一个数!");
    // 创建一个可变的字符串来存储用户输入
    let mut guess = String::new();
    // 从标准输入读取一行内容到guess字符串中
    io::stdin()
        .read_line(&mut guess) 
        .expect("读取行失败!"); // 如果读取失败,打印错误信息
    // 打印用户猜测的数字
    println!("你猜测的数字是: {}", guess);
}
```
use：引用其他库
&：表示引用,默认是不可变的
{}：是一个占位符
