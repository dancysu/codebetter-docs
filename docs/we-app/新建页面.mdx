# 新建页面

在微信小程序开发中,新建页面主要有以下两种方式：

**方式一：通过目录操作新建页面**

1. 在项目根目录下的 `pages` 文件夹上点击右键,选择「新建文件夹」。
2. 输入页面名称,例如 `list`,并按下回车键创建该文件夹。
3. 在新创建的 `list` 文件夹上再次点击右键,选择「新建Page」。
4. 输入页面名称（与文件夹名称一致,即 `list`）,无需输入任何后缀名,直接按下回车键完成创建。

----

**方式二：通过   app.json   文件新建页面**

1. 打开项目根目录下的 `app.json` 文件。

2. 在 `pages` 字段的数组中添加新的页面路径,例如：

   ```json
   {
     "pages": [
       "pages/index/index",
       "pages/list/list"
     ]
   }
   ```

3. 保存 `app.json` 文件后,微信小程序开发工具会自动根据路径生成对应的页面文件夹和文件。
