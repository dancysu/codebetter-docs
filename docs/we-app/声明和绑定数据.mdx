# 数据绑定

小程序页面中使用的数据均需要在 `Page()` 方法的 `data` 对象中进行声明定义

在将数据声明好以后,需要在 WXML 中绑定数据,数据绑定最简单的方式是使用 Mustache 语法（双大括号）将变量包起来。

在 `{{ }}` 内部可以做一些简单的运算,支持如下几种方式：

1. 算数运算

2. 三元运算

3. 逻辑判断

4. 其他…

注意：在 {{ }} 语法中,只能写表达式,不能写语句,也不能调用 js 相关的方法



## 定义数据

```js
Page({
  // 页面的初始数据
	data:{
		name:'Dancy'
	},
})
```





## 使用数据

```jsx
// 通过 {{}} 的语法把一个变量绑定到页面上,我们称之为数据绑定。

// 展示内容
<view> {{ name }} </view>

// 绑定属性值,如果需要动态绑定一个变量,属性值也需要使用双大括号进行包裹
<image src="{{ imgSrc }}" mode="widthFix"></image>

<!-- 算术运算 -->
<view>{{ id + 1 }}</view>
<view>{{ id - 1 }}</view>

<!-- 三元运算 -->
<view>{{ id === 1 ? '等于' : '不等于' }}</view>

<!-- 逻辑判断 -->
<view>{{ id === 1 }}</view>
```
