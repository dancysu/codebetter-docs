# 配置自动重启

（1）安装 nodemon

```bash
npm i nodemon -D
```

（2）修改 `package.json` 的 `start` 命令

```json
{
  "scripts": {
    "start": "nodemon --exec electron ."
  }
}
```

（3）配置 `nodemon.json` 规则

```json
{
	"ignore": ["node_modules", "dist"],
	"restartable": "r",
	"watch": ["*.*"],
	"ext": "html,js,css"
}
```

配置好以后,当代码修改后,应用就会自动重启了。
