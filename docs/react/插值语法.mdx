# 插值语法

在 React 中,花括号 `{}` 用于将 JavaScript 表达式插入到 JSX 中。

你可以插入数字、字符串、数组（普通类型）、JSX 元素、三元表达式、函数调用等各种类型的表达式。

需要注意的是,插入的内容必须是一个表达式,不能是语句,如   if   语句等。

```tsx
import './App.css'

export default function App() {
  return (
    <>
      {/* 插入数字 */}
      {/* 直接插入数字 123 */}
      <div>{123}</div>

      {/* 插入字符串 */}
      {/* 插入字符串 '456',也可以不加引号直接写 456,因为 JSX 中的文本内容会被自动识别为字符串 */}
      <div>{'456'}</div>

      {/* 插入数组 */}
      {/* 插入数组,数组中的每个元素会被转换为字符串并拼接在一起,显示为 '123' */}
      <div>{['1', 2, '3']}</div>

      {/* 插入 JSX 元素 */}
      {/* 插入一个 JSX 元素,即一个 span 标签,里面包含文本 '123' */}
      <div>{<span>123</span>}</div>

      {/* 插入三元表达式 */}
      {/* 插入一个三元表达式,根据条件的真假值显示 'yes' 或 'no',这里条件为 true,所以显示 'yes' */}
      <div>{true ? 'yes' : 'no'}</div>

      {/* 插入函数调用 */}
      {/* 插入一个函数调用,将数字 3.1415926 四舍五入到小数点后两位,显示为 '3.14' */}
      <div>{3.1415926.toFixed(2)}</div>
    </>
  )
}
```
