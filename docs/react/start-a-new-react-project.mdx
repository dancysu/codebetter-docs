# 启动一个新的 React 项目

## 使用 vite 安装 react

通过以下简单步骤,即可借助 Vite 快速搭建 React 项目：

```bash
npm init vite
```

在提示中依次输入：

- `Project name`：例如输入 `react-demo`,这是你的项目文件夹名称。

- `Select a framework`：直接选择 `React`,这是目前最主流的前端框架之一。

- `Select a variant`：推荐选择 `TypeScript + SWC`,TypeScript 能提供更强大的类型检查功能,SWC 则是下一代的 JavaScript 编译器,可大幅提升构建速度。



## 使用脚手架安装 react

如果你更倾向于传统的安装方式,可以使用以下命令：

```bash
npx create-react-app <project_name>
```

只需将 `<project_name>` 替换为你想要的项目名称,比如 `my-react-app`,即可快速创建一个 React 项目。
