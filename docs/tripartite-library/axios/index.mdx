# 介绍

:::tip Axios是前端最流行的ajax请求库，它是一个适用于浏览器和 Node.js 的基于 Promise 的 HTTP 库。

- 官网：https://axios-http.com
- Github：https://github.com/axios/axios

:::

## 安装

import { PackageManagerTabs } from '@theme';

<PackageManagerTabs command="install axios" />



## 特性

1. 从浏览器发送 `ajax` 请求
2. 从 Nodejs 发送 `http` 请求
3. 支持[Promise](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise) API
4. 拦截请求和响应
5. 转换请求和响应数据
6. 取消请求
7. 自动转换 [JSON](https://www.json.org/json-en.html) 数据
8. 自动数据对象序列化 `multipart/form-data` 和 `x-www-form-urlencoded` 主体编码
9. 客户端支持防御XSRF
